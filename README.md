# Concu-mining

## Install
 - Install cargo: curl https://sh.rustup.rs -sSf | sh

## Compile
 - cargo build

## Run
 - cargo run -- --rounds 5 --miners 12
 
 
## Executable
cargo build && cp ./target/debug/concu-mining ./concu-mining
